Convertisseur de devises en PHP

Ce projet est une application web simple en PHP permettant de convertir un montant donné (en Euro, Dollar ou Dirham) vers le Franc CFA.
Il illustre les notions de base suivantes :

Traitement de formulaires en PHP

Méthode POST

Utilisation des conditions

Affichage dynamique des résultats

Intégration de CSS pour styliser une interface utilisateur moderne

Fonctionnalités

Entrée d’un montant à convertir

Sélection d’une devise (Euro, Dollar, Dirham)

Conversion automatique vers le FCFA

Interface graphique stylisée en HTML + CSS

Résultat affiché instantanément après soumission

Technologies utilisées
Technologie	Description
HTML5	Structure du formulaire
CSS3	Mise en forme moderne et responsive
PHP 8+	Traitement du formulaire et calcul de conversion
